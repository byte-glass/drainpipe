# drainpipe

## usage

To run `./src/main.rs`,

```sh
cargo run --bin drainpipe
```

note that `drainpipe` is the project directory (name of the rust project?)

To run one (?) of the files in `./src/bin` e.g. `drain.rs`,

```sh
cargo run --bin drain
```
